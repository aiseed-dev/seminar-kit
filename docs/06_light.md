# 06 軽量キット(小規模イベント向け・システム外)

単発・小規模のイベントには本システムを使わない。**AI(Claude Code 等)に
その場で生成させた3点セット**で、担当者が自分のPCだけで完結できる。
データベース不要・サーバー不要。

## 3点セット(AI への頼み方)

「このイベントの (1) 案内ページ、(2) 申込用の神Excel、(3) 集計プログラム
を作って」と依頼する。

1. **案内ページ**: 静的 HTML 1枚(開催要領・様式のダウンロードリンク)。
   Cloudflare Pages に置くか、既存サイトの1ページとして掲載
2. **申込様式(神Excel)**: 本システムの様式と同じ設計方針
   (名前付きセル・入力規則・保護・記入例)。参加者はメール添付で返送
3. **集計プログラム**: 受信箱から保存した添付 xlsx をフォルダに入れて
   実行すると、名簿 xlsx(openpyxl)が出る Python スクリプト1本

## 使い分けの指針

| 条件 | 使うもの |
|---|---|
| 年間を通じた反復開催・定員管理・実績台帳・事前登録が要る | 本システム |
| 単発・小規模(目安: 数十件まで)・その場限り | 軽量キット |

軽量キットで始めたイベントが定例化したら、様式はそのまま本システムの
経路2(mailin)に載せ替えられる(様式の設計方針が共通のため)。

## 意味

- 担当者の自走の最小形。ITコーディネーターが AI に頼んで自分で作れる
  水準であり、引き継ぎ研修の教材そのものになる
- 生成に使う AI もローカルでよい: 集計プログラムのような定型的な
  Python スクリプトは、機関内のローカルコーディングモデル
  (North Mini Code 等+OpenCode)で生成できる(docs/05)。
  実名データを扱うコードを外に出さずに作れる
- 「規模に道具を合わせる」実例。小さな仕事に大きなシステムを
  持ち込まないこと自体が、本提案の設計思想の一部
